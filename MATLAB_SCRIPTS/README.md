# MATLAB SCRIPTS
**1. gear_calculations.m** - The main script that takes input parameters and outputs an optimal solution.  
**2. calculate_gears.m** - Function script that does all the calculations.  
**3. individual_calculation.m** - Script that allows for inputting any number of gear teeth. This was to allow modifying a solution given by the main script.  
**4. Shaft_Force_Diagram_Plots.m** - Script for generating shaft shear force, bending moment and torque diagrams.  
**5. 251004-Planet_Gear_Table.pdf** - Gear table, showing the details about each stage.
